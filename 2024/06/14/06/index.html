如何在 Visual Studio Code 中调试 Doris FE

必要插件安装

  1. Debugger for Java 插件

  2. Language Support for Java™ by Red Hat 插件

配置 launch.json 文件

关闭 VSCode 内置表达式求值功能

在 VSCode 的设置中，搜索 debug.inlineValues，将其设置为 off 即可。或直接在 settings.json 文件中添加如下配置项：

设置为 off 的理由是：内置表达式求值功能会导致调试时变量的值在代码页面上时频繁交替出现，影响调试体验。

修改Doris FE 配置文件 (fe.conf)

启动 Doris FE

启动 Doris FE 后，在 VSCode 中点击调试按钮，选择 Attach (fe) 配置项，点击开始调试按钮即可。

调试

调试时，可以设置断点，查看变量，调用堆栈等。

调试控制台，调试支持自动补全，可以直接输入方法名、类名、变量名，按下 Tab 键即可自动补全。
